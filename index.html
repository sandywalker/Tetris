How To Play

Tetris is a game wherein the goal is to completely fill rows of the grid with the falling shapes in order to accrue points. This causes the rows to be removed and all filled squares above that row to drop down one row. Each row is worth 100 base points and removing more rows increases the base points by 100 points per row removed plus a reward of: two to the power of number of rows removed minus one (i.e. three rows will get 300 base points and 400 reward points for a total of 700 points). Finally, the level speeds up the rate at which the shapes fall and increases by one every two minutes that the game is played.
